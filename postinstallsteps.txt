Manual Configuration:
====

1) Wireguard
  * get wireguard conf file
  * "nmcli connection import type wireguard file thefile.conf"
  * for gnome: manually add and enable extension
    https://extensions.gnome.org/extension/3612/wireguard-indicator/
    by atareao
    edit the settings and tell it to use nmcli

2) Systray
  * for gnome: enable the KstatusNotifier extension (already installed)
